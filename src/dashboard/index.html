Локальный OpenAI-совместимый прокси к Qwen Chat. Подставьте Base URL и любой ключ в любой клиент, который умеет OpenAI API.

Как подключить

—
—
любой непустой (например sk-qwen)
qwen3.7-max
Подходит для Open WebUI, Cherry Studio, Chatbox, LibreChat, SillyTavern, LiteLLM, OpenAI SDK и т.п. Open WebUI обычно использует URL c /api, большинство клиентов и OpenAI SDK — c /api/v1.
